# finance_scraper

Is a scraper which read finance data from website

## Installation

1. **Clone the repository:**

```bash
git clone https://github.com/J1NX5/finance_scraper
```

```bash
cd finance_scraper
```

```bash
python3 -m venv .venv
```

```bash
source .venv/bin/activate
```

```bash
pip install -r requirements.txt
```

```bash
python scraper.py
```
The scraper will create a folder named reports. The current incoming of the earning from today in there as json-file.
